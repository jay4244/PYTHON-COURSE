function categories :

There are 4 type of categories

1) function without parameter and function without return type.

    syntax : def function():
                statement

2) function with parameter and function without return type.

    synatax : def function (parameter1,parameter2..,):
                statement
 
3) function without parameter and function with return type.

    note : at a time we can return single statment or single value.
    syntax : def function():
                return statement

4) function with parameter and with return type.

    syntax : def function(parameter1,parameter2...,):
                return statement
